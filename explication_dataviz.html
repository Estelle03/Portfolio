Dataviz

Voir le site ↗

Le projet Dataviz est un projet MMI de 2ème année. Le projet était de faire un site avec un élément de visualisation de données. Avec Emy BIGIRIMANA et Jeanne BESOMBES nous avons choisis de faire une carte intéractive sur l'évolution des lois sur le droit de l'avortement en Europe.


Sur ce projet mon role à été de recolter des données sur les pays, cette taches a été la plus longue et ... qui nous a demander a nous trois notre implication.
A coté de cela, j'ai travaillé sur la structure et le design visuel du site. Combien de pages, qu'elle est leur role, les éléments qui les composerons. Les animations, les coueleurs, les typographies. Nottament avec un oeil sur l'accessibilité, en verifiant les contraste entre couleurs et textes, la tailles des éléments etc.
Le delais de réalisation de ce projet ne m'a pas permis de pousser l'accessibilité de ce site aussi loin que je l'aurais souhaité, les modifications visuelles de la carte étant seulement basée sur la couleur, cela pose un problème pour les personnes avec une difficultée pour voir les differents couleurs.
